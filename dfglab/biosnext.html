BIOS Future

What is the future of the Biogeographic Information and Observation System (BIOS)? Beyond just continuously growing biogeographic data collection and the viewer to browse for the datasets in the data catalog, what additional wildlife conservation use can we do with the library of data?

BIOS Project

  2. Library

  4. ?

BIOS Registry

The BIOS Registry catalogs every biogeographic dataset that is collected into the BIOS data library.

Next

Move the registry system to the web instead of desktop MS Access front end with a local enterprise SQL Server back end.

BIOS Data Library

The BIOS Data Library is the store house for all the biogeographic dataset collected and published to a web platform on ArcGIS Server or ArcGIS Online.

Next

Migrate the entire data collection to a cloud such as ArcGIS Online.

BIOS Viewer

The BIOS map viewer on the web is an online map viewer for search for BIOS data and use basic GIS tools to look at the data.

Issue

Keeping up with every advancing web GIS technology and APIs to maintain and enhance the in-house, custom web app is difficult. Feels like as we keep having to migrate to a new platform or new version just as the older version is just becoming stable.

Next

How to maintain an online data catalog with search interface and quick map view and also build and maintain special tools for certain datasets without having to always migrate to the next new language/platform? Utilize growing web mapping platforms and application templates
